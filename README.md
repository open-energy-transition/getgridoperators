# getgridoperators

Use WikiData to collect and refine data on transmission system operators, energy ministries, electricity distribution network operators, electricity supply companies and electric power transmission and power pools. 

Sources used for Names: 

1. [Upbeat](https://utilityperformance.energydata.info/download) 
2. [Wikipedia - Electricity distribution companies by country](https://en.wikipedia.org/wiki/Electricity_distribution_companies_by_country)
3. [Wikipedia - List of energy regulatory bodies](https://en.wikipedia.org/wiki/List_of_energy_regulatory_bodies)
4. [Wikipedia - List of United States electric companies](https://en.wikipedia.org/wiki/List_of_United_States_electric_companies)
5. [ENTSO-E Member Companies](https://www.entsoe.eu/about/inside-entsoe/members/)
6. [Global Transmission TSO Profiles and Benchmarking Report](https://globaltransmission.info/wp-content/uploads/2024/11/Global-Electricity-TSO-Profiles-and-Benchmarking-Report-2025.pdf)
